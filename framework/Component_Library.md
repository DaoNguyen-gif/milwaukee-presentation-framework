# Component Library

Version: 1.0

---

# Overview

This document defines every reusable design component used in the Master_Template.pptx.

Rules:

- Every slide MUST be built using components in this document.
- Never create a new component unless it is added here.
- Components are reusable.
- Components must keep consistent size, spacing, typography and colors.
- Components should never be stretched or distorted.

# Global Rules

Spacing

24 px grid

Margins

Top

40 px

Bottom

30 px

Left

40 px

Right

40 px

Typography

Title

Bold

Uppercase

Body

Regular

Colors

Primary

Milwaukee Red

Secondary

Black

Background

White

Icons

Always outline style.

Do not mix outline and filled icons.

Never use 3D icons.

Never use emoji.

Component Reuse

Always reuse existing components.

Never redesign.

Never invent new layouts.

Overflow Rule

If content exceeds the available space:

Create a new slide using the same layout.

Never reduce font size below the template standard.
